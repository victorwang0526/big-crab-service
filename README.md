## REST API Service

jersey + spring boot + mysql + swagger + mybatis

NEED: gradle + maven + JDK

#### mysql

```
create database carrot;

grant all privileges on *.* to carrot@localhost Identified by "carrot";
```

### start service

```
gradle bootRun
```

可访问：
http://localhost:8090/carrot
